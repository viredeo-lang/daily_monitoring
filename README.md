# 📈 Daily Monitoring App

Isang simpleng web-based application para sa pag-track ng mga daily activities o data monitoring. Ang project na ito ay binuo gamit ang HTML at naka-host via GitHub Pages.

## 🚀 Live Demo
Maaari mong ma-access ang app dito:
[https://viredeo-lang.github.io/daily_monitoring/](https://viredeo-lang.github.io/daily_monitoring/)

## ✨ Mga Tampok (Features)
* **User-friendly Interface:** Simple at madaling gamitin.
* **Web-based:** Maaaring buksan sa kahit anong browser (Chrome, Safari, Edge).
* **Mobile Responsive:** Gumagana nang maayos kahit sa mobile phones.

## 🛠️ Teknolohiya
* **HTML5** - Para sa structure ng website.
* **GitHub Pages** - Para sa libreng hosting.

## 📂 Paano Gamitin
1. I-clone ang repository o i-download ang `index.html`.
2. Buksan ang file sa iyong paboritong web browser.
3. Simulan ang pag-monitor ng iyong daily data!
